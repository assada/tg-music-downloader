# Telegram downloader bot

This bot wil download audio files which you are send to him.

### Requirements
* python 2.7
* ffmpeg

### Running
1) Install dependencies:
```bash
make build
```
2) Run:
```bash
make run "<token>" "<admin_username>" "<destination_path>"
```
